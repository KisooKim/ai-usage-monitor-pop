# AI Usage Monitor Pop — product site

Static landing page (Apple-product-page style). No build step.

A lightweight Windows widget that shows your Claude Code (and Codex) 5-hour and
7-day usage at a glance — a floating pop panel by the taskbar.

## Preview
    python -m http.server 8000
Open http://localhost:8000/

## Deploy
GitHub Pages from this repo's `main` branch root.
Served at https://kisookim.github.io/ai-usage-monitor-pop/

The Windows build is distributed as a GitHub Release asset of this repo; the
Download button links to `releases/latest/download/claude-code-usage-monitor.exe`.

Based on the open-source CodeZeno/Claude-Code-Usage-Monitor (MIT).
